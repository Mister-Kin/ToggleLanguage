## 简介
一个Blender插件，目的是通过一次点击在两种语言之间快速而方便地切换用户界面，而不是重复地打开偏好设置。

## 功能
- 一键切换用户界面语言（支持17种语言和快捷键）
- 一键打开用户偏好设置（支持快捷键）
- 一键设置Blender的个人设置
- 一键切换提示模式：默认模式和开发者模式
- 一键删除当前场景集合和物体
- 一键添加视频进度条
- ……

更多详细的功能介绍请详看[使用手册](https://mister-kin.github.io/works/software-works/toggle-language/)。
